\section{Results}

\subsection{RMSE and MAE by Algorithm}

RMSE and MAE were computed between the sample set and the predicted ratings for each algorithm:
$$\text{RMSE} = \sqrt{\frac{1}{|\mathcal{T}|}\sum_{(u,i) \in \mathcal{T}}(r_{ui} - \hat{r}_{ui})^2}$$
$$\text{MAE} = \frac{1}{|\mathcal{T}|}\sum_{(u,i) \in \mathcal{T}}|r_{ui} - \hat{r}_{ui}|$$
Table 1 gives the performance metrics for each fold across all five algorithms.
