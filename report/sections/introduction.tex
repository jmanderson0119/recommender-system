\section{Introduction}

Recommendation systems are becoming increasingly relevant in the digital age to provide personalized content on e-commerce dashboards, social media feed, and video 
streaming platforms. Our analysis compares the performance of five algorithms for recommender systems using movie ratings from the MovieLens small dataset: random, k nearest neighbors (kNN),
singular value decomposition (SVD), matrix factorization with regularization, and a deep neural network. The random algorithm serves as a baseline upon which the four collaborative
filtering algorithms can be comparatively evaluated. Each algorithm is evaluated using the same 5-fold cross-validation data partitions, and performance for each is reported as root
mean squared error (RMSE), mean absolute error (MAE), and runtime.
